# tchori

Agent-native "everything as code" engine. Speaks the Terraform plugin
protocol (tfplugin6), so existing Terraform/OpenTofu providers work day one,
while adding a structured plan API, JSON-native config, machine-readable
diagnostics, and a built-in MCP server.

Status: pre-MVP, under active development.

Design spec: `Tchori-Labs/main`,
`docs/superpowers/specs/2026-07-09-tchori-engine-design.md`.
